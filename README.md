## 必要なもの
### Autohotkey

### かえうち

職場ではインストールは当然無理なので、代用として。

## やりたいこと

### IMEのOn/Offをデフォルトのトグル式ではなく、onとOffを別々のキーに割り当てる。

### 矢印キー、home, pgUpなどをホームポジションから入力しやすくする。
具体的には独自の修飾キーとそれを利用したショートカットを作る
